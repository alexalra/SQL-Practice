How many employees have their birthday in May?

  
<details>

  <summary>Answer</summary>
  

```
2
```
Code

```ruby

SELECT
  COUNT(NEWDATE) AS NUMBER_OF_EMPLOYEES_BIRTHDAY_MAY
FROM
  (SELECT
    SUBSTRING(date(HireDate),6,2) AS NEWDATE
  FROM
    Employee
  WHERE
    NEWDATE = '05')

```
